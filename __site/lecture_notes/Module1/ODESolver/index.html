1.4 An ODE solver: the Forward Euler method

The last version of our ODE solver as seen in Truncation Errors results in the Forward Euler (FE) method for second order ODEs:

ui=ui−1+Δtu˙i−1,u˙i=u˙i−1+Δtu¨i−1,,∀i=1,...,N. \begin{align*} u_i&=u_{i-1} + \Delta t\dot{u}_{i-1},\\ \dot{u}_i&=\dot{u}_{i-1}+\Delta t\ddot{u}_{i-1}, \end{align*},\qquad\qquad \forall i=1,...,N.ui​u˙i​​=ui−1​+Δtu˙i−1​,=u˙i−1​+Δtu¨i−1​,​,∀i=1,...,N.

Remember that the value of u¨i−1\ddot{u}_{i-1}u¨i−1​ is typically given by the Equation of Motion of the system.

Equation (1) can be re-written in a vector form by defining the so called state variable

qi=[uiu˙i],\mathbf{q}_i=\left[\begin{align*}&u_i\\&\dot{u}_i\end{align*}\right],qi​=[​ui​u˙i​​],

leading to a (FE) scheme with truncation error of order O(Δt2)\mathcal{O}(\Delta t^2)O(Δt2)

qi=Δt q˙i−1,∀i=1,...,N. \mathbf{q}_i = \Delta t\ \dot{\mathbf{q}}_{i-1},\qquad\qquad \forall i=1,...,N. qi​=Δt q˙​i−1​,∀i=1,...,N.

To find the solution at all times tit_iti​ for i=1,...,Ni=1,...,Ni=1,...,N, we need the following inputs:

  • the initial conditions at t0t_0t0​, q0\mathbf{q}_0q0​

  • a function that computes q˙i\dot{\mathbf{q}}_iq˙​i​ given qi\mathbf{q}_iqi​, q˙i=f(qi)\dot{\mathbf{q}}_i=f(\mathbf{q}_i)q˙​i​=f(qi​)

  • the time step size, Δt\Delta tΔt, and the final time, TTT

Implementation

Let's see how we can implement the FE solver in a very simple example. Assume that we want to solve the mass-damper-spring system introduced at the introduction of this module

mu¨(t)+cu˙(t)+ku(t)=f(t), m\ddot{u}(t)+c\dot{u}(t)+ku(t)=f(t), mu¨(t)+cu˙(t)+ku(t)=f(t),

with the following input parameters:

const k = 1   # [N/m]
const c = 0.1 # [Ns/m]
const m = 1   # [kg]
const ξ = 0.1 # [N]
const ω = 1   # [rad]
f(t::Real) = ξ*sin(ω*t)

and the following input conditions:

const u₀ = 1.0
const v₀ = 0.0

We define a time step size and a final time: 

const Δt = 0.05
const T = 10.0

As an input, we aslo need the function that computes q˙i\dot{\mathbf{q}}_iq˙​i​ given qi\mathbf{q}_iqi​:

function dq(q,t)
    (u,v) = q              # unpack variables
    du = v                 # displacement time derivative is equal to velocity
    dv = (f(t)-k*u-c*v)/m  # acceleration from equation of motion
    dq = (du,dv)           # pack variables
end

The Forward Euler solver can be defined as a function that given the initial conditions, time step size and final time, it returns a vector with the solution:

function fe_solver(q₀,Δt,T)
    N = Int(T/Δt) + 1                           # number of time steps
    q = Array{Tuple{Float64,Float64}}(undef,N)  # Define the solution array (vector of tuples containing displacement and velocity)
    q[1] = q₀                                   # Initialize initial value
    for (i,tᵢ) in enumerate(Δt:Δt:T)            # Loop over time steps (from t=Δt to t=T with a time step size of Δt)
        q[i+1] = q[i] .+ Δt .* dq(q[i],tᵢ)      # update the vector with the FE scheme
    end
    return q                                    # return the solution vector
end
