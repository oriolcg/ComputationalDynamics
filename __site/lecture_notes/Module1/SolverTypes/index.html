1.6 Types of ODE solvers

In previous sections we have defined the Forward Euler method. We have seen that it can be defined as

qi=qi−1+Δt q˙i−1+O(Δt2). \mathbf{q}_i = \mathbf{q}_{i-1} + \Delta t\ \dot{\mathbf{q}}_{i-1} + \mathcal{O}(\Delta t^2).qi​=qi−1​+Δt q˙​i−1​+O(Δt2).

We re-write equation (1) in such a way that the value at tit_iti​ is given as a result of evaluating a function, fFEf_{\text{\tiny FE}}fFE​, that takes as arguments Δt\Delta tΔt, qi−1\mathbf{q}_{i-1}qi−1​ and q˙i−1\dot{\mathbf{q}}_{i-1}q˙​i−1​:

qi=fFE(Δt,qi−1,q˙i−1)+O(Δt2). \mathbf{q}_i = f_{\text{\tiny FE}}(\Delta t,\mathbf{q}_{i-1},\dot{\mathbf{q}}_{i-1}) + \mathcal{O}(\Delta t^2).qi​=fFE​(Δt,qi−1​,q˙​i−1​)+O(Δt2).

In the case we have an equation of motion that relates u¨i\ddot{u}_{i}u¨i​ with uiu_iui​ and u˙i\dot{u}_{i}u˙i​, we can simplify even more the function fFEf_{\text{\tiny FE}}fFE​, so it only takes as arguments Δt\Delta tΔt and qi−1\mathbf{q}_{i-1}qi−1​.

qi=fFE(Δt,qi−1)+O(Δt2). \mathbf{q}_i = f_{\text{\tiny FE}}(\Delta t,\mathbf{q}_{i-1}) + \mathcal{O}(\Delta t^2).qi​=fFE​(Δt,qi−1​)+O(Δt2).

Depending on the definition of the function fff, one can classify the ODE solver in different categories:

  • Implicit vs Explicit

  • Single-step vs Multi-step

  • Single-stage vs Multi-stage

Implicit and Explicit solvers

We say that an ODE solver is explicit when the function that gives the state vector at time tit_iti​ only depends on the state vector and its derivatives at previous times. That is

qi=fexplicit(Δt,qj,q˙j,...,qj−1,q˙j−1,...)for j<i. \mathbf{q}_i = f_{\text{\tiny explicit}}(\Delta t,\mathbf{q}_{j},\dot{\mathbf{q}}_{j},...,\mathbf{q}_{j-1},\dot{\mathbf{q}}_{j-1},...)\qquad \text{for }j < i.qi​=fexplicit​(Δt,qj​,q˙​j​,...,qj−1​,q˙​j−1​,...)for j<i.

Note that the Forward Euler method is an explicit method, since fFE(Δt,qi−1)f_{\text{\tiny FE}}(\Delta t,\mathbf{q}_{i-1})fFE​(Δt,qi−1​) only depends on the state vector at ti−1t_{i-1}ti−1​.

We say that an ODE solver is implicit when the function that gives the state vector at time tit_iti​ depends on the same state vector at time tit_iti​. That is

qi=fimplicit(Δt,qj,q˙j,...,qj−1,q˙j−1,...)for j<=i. \mathbf{q}_i = f_{\text{\tiny implicit}}(\Delta t,\mathbf{q}_{j},\dot{\mathbf{q}}_{j},...,\mathbf{q}_{j-1},\dot{\mathbf{q}}_{j-1},...)\qquad \text{for }j <= i.qi​=fimplicit​(Δt,qj​,q˙​j​,...,qj−1​,q˙​j−1​,...)for j<=i.

An example of implicit solver is the Backward Euler scheme, that computes the state vector at tit_iti​ as

qi=fBE(Δt,qi˙,qi−1)+O(Δt2)=qi−1+Δt q˙i+O(Δt2). \mathbf{q}_i = f_{\text{\tiny BE}}(\Delta t,\dot{\mathbf{q}_i},\mathbf{q}_{i-1}) + \mathcal{O}(\Delta t^2) = \mathbf{q}_{i-1} + \Delta t\ \dot{\mathbf{q}}_{i} + \mathcal{O}(\Delta t^2).qi​=fBE​(Δt,qi​˙​,qi−1​)+O(Δt2)=qi−1​+Δt q˙​i​+O(Δt2).

Single-step and multi-step solvers

We say that an ODE solver is a single-step solver when the function that gives the state vector at time tit_iti​ only depends on the state vector and its derivatives at tit_iti​ or ti−1t_{i-1}ti−1​. That is

qi=f1-step(Δt,qi,qi−1,q˙i,q˙i−1,...). \mathbf{q}_i = f_{\text{\tiny 1-step}}(\Delta t,\mathbf{q}_{i},\mathbf{q}_{i-1},\dot{\mathbf{q}}_{i},\dot{\mathbf{q}}_{i-1},...).qi​=f1-step​(Δt,qi​,qi−1​,q˙​i​,q˙​i−1​,...).

Note that the Forward Euler and Backward Euler methods are single-step methods.

We say that an ODE solver is a multi-step solver when the function that gives the state vector at time tit_iti​ depends on the state vector and its derivatives at multiple previous times. That is

qi=fn-step(Δt,qi,qi−1,q˙i,q˙i−1,...,qj,q˙j,...,qj−1,q˙j−1,...)for j<i−1. \mathbf{q}_i = f_{\text{\tiny n-step}}(\Delta t,\mathbf{q}_{i},\mathbf{q}_{i-1},\dot{\mathbf{q}}_{i},\dot{\mathbf{q}}_{i-1},...,\mathbf{q}_{j},\dot{\mathbf{q}}_{j},...,\mathbf{q}_{j-1},\dot{\mathbf{q}}_{j-1},...)\qquad \text{for }j < i-1.qi​=fn-step​(Δt,qi​,qi−1​,q˙​i​,q˙​i−1​,...,qj​,q˙​j​,...,qj−1​,q˙​j−1​,...)for j<i−1.

An example of multi-step scheme are the Backward Differentiation Formulas. For instance, the second-order Backward Differentiation Formula (BDF2) reads:

qi=fBDF2(Δt,q˙i,qi−1,qi−2)=43qi−1−13qi−2+23Δtq˙i. \mathbf{q}_i =f_{\text{\tiny BDF2}}(\Delta t,\dot{\mathbf{q}}_{i},\mathbf{q}_{i-1},\mathbf{q}_{i-2}) = \frac{4}{3}\mathbf{q}_{i-1}-\frac{1}{3}\mathbf{q}_{i-2} + \frac{2}{3}\Delta t\dot{\mathbf{q}}_{i}.qi​=fBDF2​(Δt,q˙​i​,qi−1​,qi−2​)=34​qi−1​−31​qi−2​+32​Δtq˙​i​.

Single-stage and multi-stage

We say that an ODE solver is a single-stage solver when the function that gives the state vector at time tit_iti​ depends on the state vector and its derivatives at previous steps, but it does not use any additional intermediate stages.

The Forward Euler, Backward Euler and BDF schemes are single-stage solvers.

We say that an ODE solver is a multi-stage solver when the function that gives the state vector at time tit_iti​ only depends on the state vector and its derivatives at tit_iti​ or ti−1t_{i-1}ti−1​, but it uses additional intermediate stages.

An example of multi-stage solvers are the Runge-Kutta methods. For example, the fourth-order Runge-Kutta scheme (RK4) can be defined as:

qi=qi−1+16Δt(k1+2k2+2k3+k4),k1=q˙i−1,k2=f(Δt2,qi−1+Δtk12),k3=f(Δt2,qi−1+Δtk22),k4=f(Δt,qi−1+Δtk3). \begin{align*} \mathbf{q}_i =& \mathbf{q}_{i-1} + \frac{1}{6}\Delta t(k_1+2k_2+2k_3+k_4),\\ k_1=&\dot{\mathbf{q}}_{i-1},\\ k_2=&f\left(\frac{\Delta t}{2},\mathbf{q}_{i-1}+\Delta t\frac{k_1}{2}\right),\\ k_3=&f\left(\frac{\Delta t}{2},\mathbf{q}_{i-1}+\Delta t\frac{k_2}{2}\right),\\ k_4=&f\left(\Delta t,\mathbf{q}_{i-1}+\Delta tk_3\right). \end{align*} qi​=k1​=k2​=k3​=k4​=​qi−1​+61​Δt(k1​+2k2​+2k3​+k4​),q˙​i−1​,f(2Δt​,qi−1​+Δt2k1​​),f(2Δt​,qi−1​+Δt2k2​​),f(Δt,qi−1​+Δtk3​).​
