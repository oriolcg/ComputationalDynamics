2.2. Lagrangian Mechanics

In a wide variety of applications, we are interested on defining the Equations of Motion (EoM) of a system that might be composed by several elements. The compatibility of motions between these elements introduce certain constraints. Then, determining the EoM of the full system can result in a convoluted task.

Luckily, there is an approach that allows us to derive the EoM of complex systems in a procedural manner: the Euler-Lagrange approach.

What is a Lagrangian

First, let us start by defining the concept of Lagrangian, L\mathcal{L}L, which is given by the following expression

L=T−V.\mathcal{L}=T-V.L=T−V.

Here, TTT is the kinetic energy of the system and VVV the potential energy.

Knowing its definition, one can compute it by following these steps:

  1. Define the kinematic relations: That means to describe the location of each relevant point using the generalized coordinates of the system (GCS)

    • The relevant points are on the points that have mass, centre of gravity of a rigid body, connection points, etc.

    • If multiple points are connected to each other, it is easier to express the coordinates relative to each other.

  2. Define the energy equations: That means to write down the kinetic and potential energy of all the elements

  3. Replace the kinematic equations into the energy equations

  4. Done!

Equations of motion from a Lagrangian

Once we have the definition of the Lagrangian, L\mathcal{L}L, we can compute the system EoM by using the Lagrange's equation:

ddt∂L∂g˙i=∂L∂gifor i=1,...,nDOF, \frac{d}{dt}\frac{\partial\mathcal{L}}{\partial\dot{g}_i}=\frac{\partial\mathcal{L}}{∂ g_i}\quad\text{for }i=1,...,nDOF,dtd​∂g˙​i​∂L​=∂gi​∂L​for i=1,...,nDOF,

where gig_igi​ is the vector of generalized coordinates of the system and g˙i\dot{g}_ig˙​i​ the corresponding generalized velocities. Note that equation (2) is a system of equations, one equation for each coordinate (or degree of freedom (DOF)).

In practice, gig_igi​ contains the DOFs of the system for i=1,...,nDOFi=1,...,nDOFi=1,...,nDOF. Therefore, the state vector q\mathbf{q}q that we used in the previous module, can be written as

q=[gig˙i].\mathbf{q}=\left[\begin{align*}&g_i\\&\dot{g}_i\end{align*}\right].q=[​gi​g˙​i​​].

Thus, to obtain the EoM we will have to follow these steps:

  1. Compute the Lagrangian

  2. Compute derivatives with respect to gig_igi​ and g˙i\dot{g}_ig˙​i​

  3. Obtain the equation of motion by replacing in the Lagrange's equation (2)

For large systems, it is difficult to derive manually the derivatives of the Lagrangian with respect to the degrees of freedom. In such cases, it is useful to make use of tools that can derive them for us. Here we can use Maple and symbolic tools from Matlab or Python. In these lecture notes we use the Julia package Symbolics.jl

An example

Let's consider the example of a vertical pendulum attached to a spring where we neglect the effect of gravity

To derive the EoM of this system we start by computing the Lagrangian, following the steps descrived earlier:

  1. Define the kinematic relations:

Assuming that p1p_1p1​ and p3p_3p3​ are known fixed points (i.e. no velocities at these points), and r1r_1r1​, mmm and kkk are known constans, the position of the point mass will be defined by the angle φ\varphiφ, which is the only variable of the system. 

using Symbolics
@variables t, ϕ(t), p₁, p₃, k, r₁, m
Dt = Differential(t)
p₂ = p₁ .+ r₁ .* [cos(ϕ) sin(ϕ)]'
p₂ₜ = Dt.(p₂)
@show p₂
@show p₂ₜ
p₂ = Symbolics.Num[p₁ + r₁*cos(ϕ(t)); p₁ + r₁*sin(ϕ(t))]
p₂ₜ = Symbolics.Num[Differential(t)(p₁ + r₁*cos(ϕ(t))); Differential(t)(p₁ + r₁*sin(ϕ(t)))]

  1. Define the energy equations:

using LinearAlgebra
T = 1/2*m*norm(p₂ₜ)^2
r₂₀ = norm(p₁.-p₃)-r₁
r₂ = norm(p₂.-p₃)
V = 1/2*k*(r₂-r₂₀)
@show T
@show V
T = 0.5m*(sqrt(abs2(Differential(t)(p₁ + r₁*cos(ϕ(t)))) + abs2(Differential(t)(p₁ + r₁*sin(ϕ(t)))))^2)
V = 0.5k*(r₁ + sqrt(abs2(p₁ + r₁*cos(ϕ(t)) - p₃) + abs2(p₁ + r₁*sin(ϕ(t)) - p₃)) - abs(p₁ - p₃))

  1. Define the Lagrangian:

L = T-V
@show L
L = 0.5m*(sqrt(abs2(Differential(t)(p₁ + r₁*cos(ϕ(t)))) + abs2(Differential(t)(p₁ + r₁*sin(ϕ(t)))))^2) - 0.5k*(r₁ + sqrt(abs2(p₁ + r₁*cos(ϕ(t)) - p₃) + abs2(p₁ + r₁*sin(ϕ(t)) - p₃)) - abs(p₁ - p₃))

Once we have the Lagrangian definition, we appy the Lagrange equation to derive the equation of motion. To do that, we will use the Differential operator of the Symbolics.jl package:

Dϕ = Differential(ϕ)
Dϕₜ = Differential(ϕₜ)
∂L_∂ϕ = Dϕ(L)
∂L_∂ϕₜ = Dϕₜ(L)
EoM = Dt(∂L_∂ϕₜ) - ∂L_∂ϕ ~ 0
@show simplify(expand_derivatives(EoM))
simplify(expand_derivatives(EoM)) = 0.5m*(2(r₁^2)*(cos(ϕ(t))^2)*Differential(t)(Differential(t)(ϕ(t))) + 2(r₁^2)*(sin(ϕ(t))^2)*Differential(t)(Differential(t)(ϕ(t)))) + 0.25k*(r₁*(2p₁ + 2r₁*sin(ϕ(t)) - 2p₃)*cos(ϕ(t)) - r₁*(2p₁ + 2r₁*cos(ϕ(t)) - 2p₃)*sin(ϕ(t)))*(sqrt(abs2(p₁ + r₁*cos(ϕ(t)) - p₃) + abs2(p₁ + r₁*sin(ϕ(t)) - p₃))^-1) ~ 0

Now let's evaluate the model for a specific set of parameters:

EoM_expanded = expand_derivatives(EoM)
EoM_eval = substitute(simplify(EoM_expanded), (Dict( p₁=>[0.0 3.0]', p₃=>[0.0 0.0]', k=>1.0, r₁=>1.5, m=>1.0 )))
@show EoM_expanded
