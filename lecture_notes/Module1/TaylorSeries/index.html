1.1 Taylor Series

Before going into more details on how to find a numerical approximation, let's start by refreshing some theory about Taylor series. As one can find in the Wikipedia page:

the Taylor series of a function is an infinite sum of terms that are expressed in terms of the function's derivatives at a single point.

That is

f(x)=∑i=0∞f(i)(a)i!(x−a)i. f(x)=\sum_{i=0}^\infty\frac{f^{(i)}(a)}{i!}(x-a)^i. f(x)=i=0∑∞​i!f(i)(a)​(x−a)i.

The series is exact for an arbitrary function as long as we include infinite terms in the sum. However, here we are interested on an approximation that includes only the first rrr terms of the expansion:

fr(x)=∑i=0rf(i)(a)i!(x−a)i≈f(x). f_r(x)=\sum_{i=0}^r\frac{f^{(i)}(a)}{i!}(x-a)^i\approx f(x). fr​(x)=i=0∑r​i!f(i)(a)​(x−a)i≈f(x).

Let's see how this work in a practical example. We consider here the function f(x)=sin⁡(x)f(x)=\sin(x)f(x)=sin(x) and we want to approximate this function knowing the function value and its derivatives at the point a=0a=0a=0. Using five terms in the expansion, r=5r=5r=5, we have that

f5(x)=sin⁡(0)+xcos⁡(0)−x22sin⁡(0)−x36cos⁡(0)+x424sin⁡(0)+x5120cos⁡(0)=x−x36+x5120.\begin{align*}f_5(x) &= \sin(0) + x\cos(0) - \frac{x^2}{2}\sin(0) - \frac{x^3}{6}\cos(0) + \frac{x^4}{24}\sin(0) + \frac{x^5}{120}\cos(0)\\ &= x - \frac{x^3}{6} + \frac{x^5}{120}. \end{align*} f5​(x)​=sin(0)+xcos(0)−2x2​sin(0)−6x3​cos(0)+24x4​sin(0)+120x5​cos(0)=x−6x3​+120x5​.​

We now can do a first coding exercise with this example and see how this approximation looks like for different values of rrr. To do so, we explicitly define the approximated functions.

const a = 0
f(x) = sin(x)
f₀(x) = sin(a)
f₁(x) = f₀(x) + (x-a)*cos(a)
f₂(x) = f₁(x) - (x-a)^2/2 * sin(a)
f₃(x) = f₂(x) - (x-a)^3/6 * cos(a)
f₄(x) = f₃(x) + (x-a)^4/24 * sin(a)
f₅(x) = f₄(x) + (x-a)^5/120 * cos(a)

And we plot them. Here we use the package Plots.jl that contains visualization tools and LaTeXStrings.jl which is a convinient tool to use LaTeX\LaTeXLATE​X syntax in the plots.

using Plots
using LaTeXStrings
x = -2π:0.01:2π
plt = plot(x,f.(x),
           linecolor=:black,
           linewidth=3,
           label=L"\sin(x)",
           xlabel=L"x",
           ylabel=L"f(x)",
           xlim=[-2π,2π],
           ylim=[-1.5,1.5])
plot!(plt,x,f₀.(x),line=:dash,label=L"f_0(x)")
plot!(plt,x,f₁.(x),linewidth=2,line=:solid,label=L"f_1(x)")
plot!(plt,x,f₂.(x),line=:dash,label=L"f_2(x)")
plot!(plt,x,f₃.(x),linewidth=2,line=:solid,label=L"f_3(x)")
plot!(plt,x,f₄.(x),line=:dash,label=L"f_4(x)")
plot!(plt,x,f₅.(x),linewidth=2,line=:solid,label=L"f_5(x)")

We see that as we increase the number of terms, the approximation gets closer to the analytical expression. Note also that for this particular example, there is no gain for r=2r=2r=2 and r=4r=4r=4 with respect to r=1r=1r=1 and r=3r=3r=3. This is caused by the fact that sin⁡(a)=0\sin(a)=0sin(a)=0 at the approximation point a=0a=0a=0.
